## STL概述

STL 是 `Standard Template Library` 的缩写，即**标准模板库**。

STL 是C++的一部分，不用单独安装。

> C++ 对模板（Template）支持得很好，STL 就是借助模板把常用的数据结构及其算法都实现了一遍，并且做到了数据结构和算法的分离。
>
> 例如：`vector` 的底层为顺序表（数组）；`list` 的底层为双向链表；`deque` 的底层为循环队列；`set` 的底层为红黑树；`hash_set` 的底层为哈希表。
